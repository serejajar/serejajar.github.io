Что можно улучшить
- Вместо Math.round можно использовать форматирование валюты из Intl:
const formattedPrice = new Intl.NumberFormat('ru-RU').format(price);
Это добавит пробелы (1 200 ₽ вместо 1200 ₽) и число будет выглядеть более читабельно, особенно если число очень большое.

- Расчёт цены со скидкой нужон округлять (иначе будут копейки типа 1999.999999)
const discountedPrice = Math.round(price * (1 - discount));
